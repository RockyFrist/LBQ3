# 冷兵器战斗系统设计文档

> 版本：v0.21.0
> 核心定位：以体力博弈为核心，多武器差异化战斗，闭环克制
> 设计参考：只狼（架势/弹刀节奏）+ 荣耀战魂（轻重攻击/变招佯攻）+ 异人之下（闪避机动性/普攻克招架）
> 技术栈：Vite 6.x + HTML5 Canvas 2D + Vanilla JS ES Modules
> 视角：纯俯视2D，圆形色块 + 朝向指示线
> 输入支持：键鼠 / 手柄（Xbox） / 触屏虚拟按键

## 一、操作方案

### 键鼠操作

| 操作 | 按键 | 说明 |
|------|------|------|
| 移动 | WASD | 八方向移动 |
| 朝向 | 鼠标 | 角色朝向跟随鼠标 |
| 轻击 | **左键单击** | 出手快，可连按组成连击串（武器决定段数） |
| 重击 | **右键单击** | 蓄力攻击，伤害高，大部分武器附带霸体 |
| 招架 | **空格按住** | 触发格挡判定，松手后有后摇 |
| 闪避 | Shift + 方向 | 消耗1体力，朝指定方向翻滚 |
| 变招 | 攻击中按 其他操作键 | 消耗2体力，可在轻击/重击/招架间切换 |
| 绝技 | **F** | 炁满100时释放武器专属绝技 |

### 手柄操作（Xbox）

| 操作 | 按键 | 说明 |
|------|------|------|
| 移动+朝向 | 左摇杆 | 移动同时控制朝向 |
| 轻击 | X | 轻攻击 |
| 重击 | Y | 重攻击 |
| 招架 | B | 格挡 |
| 闪避 | RB / A / LB | 翻滚闪避 |
| 绝技 | RT | 释放绝技 |

### 触屏操作

- 左侧虚拟摇杆（半径55px）控制移动
- 右侧5按钮簇：轻击/重击/闪避/格挡/绝技
- 右上角返回按钮

### 变招详细

| 操作 | 效果 | 消耗 |
|------|------|------|
| 轻击中按右键 | 轻击→重击 | 2体力 |
| 轻击中按空格 | 轻击→招架 | 2体力 |
| 重击中按左键 | 重击→轻击 | 2体力 |
| 重击中按空格 | 重击→招架 | 2体力 |
| 攻击中按Shift | 闪避取消（不算变招） | 1体力 |
| 命中后 | 不可变招 |  |

## 二、游戏模式

### 对战模式（pvai）
- 玩家 vs AI，键鼠/手柄/触屏操作
- 进入前选择武器、护甲和AI难度（1-5）
- 游戏中可按数字键随时切换难度

### 斗蛐蛐（spectate — 观战模式）
- AI vs AI 观战模式
- 选择双方AI难度和武器/护甲，观看对决
- 战斗结束后胜者会远离败者（胜利行走）

### 江湖行（jianghu）
- PvE 闯关模式，连续挑战不同配置的敌人
- 关卡配置参见 `jianghu-stages.js`

### 武林大会（arena）
- 16人单淘汰锦标赛
- 生成随机命名角色，赢者获称号
- 含下注系统、解说、败者复活混战
- 最终冠军获封"武林盟主"

### 田忌赛马（horseracing）
- 策略对战模式，Best-of-N 系列赛
- 玩家分配出战顺序与AI队伍对决
- 基于武器克制、难度、护甲的胜率预估
- 5阶段晋升：郡城擂台 → 州府竞技 → 大区联赛 → 四方英雄会 → 武林总决赛
- 奖励目标：零封(3-0)、爆冷、逆袭、速战

### 连战（chainKill）
- 连续击杀生存模式

### 宗门风云（sect）
- 独立经营管理模式（详见第十四节）

### 自由训练（training）
- 场地空旷无敌人，按 E 召唤敌人，I 召唤队友
- 按 O 充满炁，R 重置战场，1-5 切换难度
- 适合练习基本操作和绝技释放

### 教学模式（tutorial）
- 分步引导学习，含子任务
- 被动/重击/格挡模式假人练习
- 进度追踪

### 本地双人（local2p）
- 手柄支持的本地双人对战

### 联机对战（online）
- WebSocket 多人对战，主机/客机模式

### 武圣/NN 挑战（wusheng）
- 对抗神经网络控制的AI
- 可对战或观战 NN vs D5

### 自动测试
- 批量AI对战数据统计
- **视觉测试**：x8倍速可视化，可观察过程
- **纯数据测试**：瞬间跑完数百轮，只看统计结果
- 选择双方难度、武器和轮数

### 无头测试（Node.js）
```
node test-runner.js --rounds 100 --diffA 5 --diffB 5        # 文本输出
node test-runner.js --rounds 300 --diffA 5 --diffB 5 --json  # JSON输出
node test-runner.js --rounds 50 --detail                     # 每轮明细
```

## 五、招架与格挡

### 招架基础
- 按住空格进入招架姿态
- 招架中暂停体力恢复2.5秒
- 松手后延迟0.12秒 + 0.55秒后摇 = 共0.67秒不可行动
- 招架状态下无法闪避

### 格挡判定（三级精度）

| 等级 | 窗口 | 自身消耗 | 敌方体力扣除 | 冻结帧 | 慢动作 |
|------|------|---------|-------------|--------|--------|
| **精准格挡** | 攻击命中前0.12秒 | 0 | 2 | 0.18s | x0.12 持续1.40s |
| **半精准格挡** | 已在招架中 | 0 | 1 | 0.12s | x0.15 持续1.10s |
| **非精准格挡** | 时机偏差大 | 2（自身） | 1（对方） | 0.06s | x0.30 持续0.50s |

### 格挡反击（Parry Counter）

精准格挡触发自动反击（部分武器专属）：

| 参数 | 数值 |
|------|------|
| 反击伤害 | 15 |
| 射程 | 65px |
| 弧度 | 63° |
| 判定时间 | 0.12s |
| 后摇 | 0.30s |
| 前冲/推进 | 30px / 120px/s |
| 连续弹反衰减 | x0.4 |

### 格挡后二次博弈

格挡成功后双方进入慢动作，**格挡方短暂停顿后先行解锁，被弹方稍后解锁**，形成有利但不绝对的博弈窗口：

| 参数 | 说明 |
|------|------|
| 攻击方被弹硬直 | 0.10游戏秒（精准格挡下约0.83真实秒） |
| 格挡方行动延迟 | 精准0.12s / 半精准0.22s / 非精准0.38s |
| 格挡加速增益 | 格挡方下次攻击前摇压缩（精准x0.80/半精准x0.85/非精准x0.90） |
| 增益持续 | 精准/半精准0.80秒，非精准0.60秒 |

**博弈选择**：
- 格挡方：利用加速增益出轻击（安全）或重击（高风险高收益）
- 被弹方：格挡（防御）、闪避（重置）、攻击（赌对方不出招）
- 格挡方有约0.5真实秒的先手优势，但不保证命中

### 格挡防误触
- 格挡成功后自动抑制Space输入（blockSuppressed），防止误入blocking
- 松开Space后恢复正常格挡能力

### 格挡拉近
| 等级 | 拉近距离 | 滑动时间 |
|------|---------|---------|
| 精准 | 35px | 0.20s |
| 半精准 | 25px | 0.20s |
| 非精准 | 0px | - |

- 拉近在武器弹开动画结束后执行（延迟拉近）
- 最小间距保护：约52px，防止贴脸

## 六、核心克制链

```
重击(霸体) -> 克制 -> 轻击(连击) -> 克制 -> 招架(格挡) -> 克制 -> 重击
闪避 -> 第四维度：重置局面 / 完美闪避
```

### 重击克轻击
- 霸体无视轻击干扰，蓄力中不会被轻击中断
- 重击命中后对方吃高伤害+大击退
- **注意：0体力时重击失去霸体**，会被轻击打断

### 轻击克招架（渐进破防）
| 连击段 | 打到招架方的效果 |
|--------|----------------|
| 第1–3下 | 招架方-1体力/下，不破防 |
| **第4下** | **破防！** 对方进入0.50秒硬直 |

> 注：破防所需次数因武器而异（刀4下，匕首3下，剑盾5下）

### 招架克重击
- 格挡重击后扣对方体力（按精度等级）
- 进入慢动作二次博弈

## 七、闪避系统

| 参数 | 数值 |
|------|------|
| 消耗 | 1体力 |
| 持续 | 0.36秒 |
| 速度 | 370 px/s |
| 无敌帧 | 前0.18秒 |
| 完美闪避窗口 | 0.08秒（无敌期前44%） |

### 完美闪避
- 闪避前0.08秒内恰好躲过攻击则触发完美闪避
- 退还闪避消耗 + 额外回1体力（净回1点）
- 可自由追击反打

### 完美闪避 vs 精准格挡

| 对比 | 精准格挡 | 完美闪避 |
|------|---------|---------|
| 体力成本 | 0 | 0（退还+1） |
| 收益 | 敌方-2体力+加速反击 | 净回1体力+自由追击 |
| 后续 | 慢动作二次博弈 | 回到中立/追击 |
| 定位 | 正面刚，连续拆招 | 侧面切，拿确定伤害 |

| 阶段 | 持续 | 说明 |
|------|------|------|

| 连斩(active) | 0.60s | 霸体，前冲120px/s，4段×7伤害=28总伤 |
| 收刀(recovery) | 0.35s | 无霸体，可被惩罚 |

### 连斩参数（刀基准）
| 参数 | 数值 |
|------|------|
| 射程 | 90px |
| 扇形角度 | 120° |
| 连斩段数 | 4段 |
| 每段伤害 | 7（格挡减伤50%→每段3.5） |
| 末段击退 | 60px |
| 前冲速度 | 120px/s |
| 多目标衰减 | 2人×0.80 / 3+人×0.66 |

### 命中锁定机制
- 首段命中后目标被锁定（`ultimateLocked`），无法闪避脱离后续连斩
- 非末段命中施加0.18秒硬直 + 15px微击退（覆盖0.15秒段间隔）
- 末段命中施加0.40秒硬直 + 60px大击退，同时解除锁定
- 首段未命中则全程可闪避（穿透格挡但不穿透闪避无敌帧）

### 格挡减伤
- 绝技穿透格挡，但格挡状态下伤害减半（每段7→3.5，总计约14）
- 被绝技抓住时来不及闪避可举盾硬接，降低损失
- 格挡减伤时闪蓝色光（`#89f`），区别于正常命中白光

### 霸体规则
- **蓄势阶段无霸体**：任何攻击均可打断，触发50%炁惩罚
- **连斩阶段有霸体**：轻击完全无效（火花特效），重击造成伤害但不打断
- **收刀阶段无霸体**：可自由惩罚
- **对策**：闪避至身后，等收刀惩罚；或用重击在不打断的情况下削血

### 绝刀相撞（绝技 vs 绝技）
- 双方同时处于绝技连斩(active)且面向对方时触发
- 双方弹开70px + 0.50秒硬直
- 双方各返还50%炁（与被打断同等惩罚）
- 特殊视觉效果：大火花 + "绝刀相撞!" 浮字 + 0.25s冻结帧 + x0.10慢动作

### 最佳使用时机
- **对手硬直中**：被打硬直无保护，绝技穿透一切 → 稳定32伤害（D3+AI会利用）
- **对手后摇中**：轻击/重击后摇无法取消 → 安全释放
- **对手格挡后摇**：格挡后有0.67s不可行动 → 绝技穿透格挡
- **中距离开局**：对手反应窗口仅400ms，贴脸更难反应
- **避免**：对手idle时正面释放（容易被打断startup）；对手也炁满时（可能绝刀相撞）

## 九、特殊机制

### 拼刀（轻击 vs 轻击同时命中）
- 双方武器弹回，推开18px
- 硬直0.25秒
- 冻结帧0.15秒
- 慢动作x0.30 持续0.35秒
- 拼刀后立即回到决策点，可形成"叮叮叮"节奏交锋

### 弹刀（重击 vs 重击碰撞）
- 双方各扣1体力
- 弹开45px
- 硬直0.40秒
- 冻结帧0.20秒
- 慢动作x0.20 持续0.50秒

### 处决
- 条件：体力=0时被攻击命中
- 伤害：最大血量的25%（25HP）
- 持续：1.0秒动画
- 处决后：被处决者恢复满体力，处决者不恢复
- **0体力的重击没有霸体**，因此对手可以用轻击打断你的重击蓄力并触发处决

### 预输入缓冲系统
- 在攻击判定、闪避、硬直、被弹、格挡后摇等锁定状态中，输入会被缓冲
- 缓冲窗口：0.35游戏秒
- 回到idle时自动消费缓冲，实现无缝衔接
- 连击链也支持缓冲输入（不会因为按早而断连）

- **被击后防御意识**：出硬直后短暂观望（概率随难度提升），打破互砍死循环
- **攻击承诺系统**：高难度AI能读招取消攻击
- **绝技使用**：炁满时AI有概率释放绝技，在合适距离内释放
- **绝技反应**：AI会在对手绝技蓄势/连斩阶段侧向闪避
- **胜利行为**：战斗结束后胜者远离败者，模拟真人感

## 十二、新手指南

### 核心克制关系（先记住这三条）

```
重击 → 打穿轻击（霸体硬扛）
轻击连击 → 打穿格挡（连续命中破防）
格挡 → 克制重击（格反获得反击窗口）
闪避 → 万能重置（但消耗体力）
```

### 体力管理（最重要的资源）

1. **体力=命**：体力归零后速度减半、可被处决、**重击失去霸体**
2. **不要乱变招**：变招消耗2体力，频繁变招会快速耗空体力
3. **不要长时间举盾**：招架会暂停体力恢复2.5秒
4. **完美闪避回体力**：精确时机的闪避不仅免费还+1体力
5. **被处决后满体力**：被处决不是世界末日，会恢复全部体力

### 从哪个难度开始？

| 难度 | 适合谁 | AI特点 |
|------|--------|--------|
| **1 新手** | 第一次玩 | 几乎不会防御，站着让你打，练习基本操作 |
| **2 普通** | 熟悉操作后 | 偶尔格挡和闪避，有破绽可抓 |
| **3 熟练** | 理解克制链 | 会走位试探、变招、识别你的攻击类型 |
| **4 困难** | 想要挑战 | 复杂变招连招、利用格挡增益反击 |
| **5 大师** | 追求极限 | 接近完美反应、高频变招、精准闪避 |

### 进阶技巧

- **格挡只对重击有收益**：轻击打格挡只扣1体力，不值得为轻击举盾
- **观察蓄力**：看到红色蓄力圈就举盾，精准格挡可以扣对方2体力
- **第3下轻击是承诺**（刀）：前摇更长、后摇巨大，但能破防，要看准时机
- **0体力别蓄重击**：没霸体的重击就是送人头
- **利用格挡后加速**：精准格挡后你的下次攻击前摇缩短20%，趁机反击

### Fighter 状态机
```
idle → lightAttack → (combo chain) → idle
idle → heavyAttack → (startup → active → recovery) → idle
idle → blocking → (linger) → blockRecovery → idle
idle → dodging → idle
idle → ultimate → (startup → active → recovery) → idle
blocking → parryCounter → idle (精准格挡自动反击)
blocking → parryStunned → idle (被弹恢复，可预输入)
any → staggered → idle (被打硬直，可预输入)
any → executing / executed (处决动画)
```

- **变招系统**：攻击中可消耗2体力切换动作
- **预输入缓冲**：0.25秒窗口，锁定状态中的输入自动排队
- **朝向锁定**：部分特殊动作（如匕首影步）锁定朝向
- **击退滑行**：0.12秒持续，线性减速

### 格挡后时序流程
```
重击命中格挡
  ↓
hitFreeze (双方冻结，玩家输入缓冲)
  ↓
慢动作开始 (x0.12~x0.30)
  ↓
格挡方: parryActionDelay 0.04游戏秒 → 解锁行动 (有parryBoost加速)
攻击方: parryStunned 0.10游戏秒 → idle (可预输入)
  ↓
二次博弈决策点 (格挡方略早解锁，但不保证命中)
```

## 十四、宗门风云（管理经营模式）

### 简介

**宗门风云**是独立的经营管理模式，脱离实时战斗，专注于宗门建设与弟子培养。玩家以门主身份，招募弟子、派遣任务、修缮设施、参与事件，逐步提升门派声望。

### 进入方式

在主菜单选择「宗门风云」，输入门派名称后开始。支持3个存档槽。

### 核心循环

```
每日推进
  ├── 随机事件触发（稀有事件、大型事件、江湖消息）
  ├── 招募新弟子（若宿舍空闲）
  ├── 任务刷新（每天3–5个任务）
  └── 银庄被动收入

管理行动（无时间限制）
  ├── 全体训练 → 所有弟子消耗体力换取经验
  ├── 专训（弟子详情）→ 单人高强度训练（-20体力 +高经验）
  ├── 派遣任务 → 选弟子出战，观看实时战斗，战斗结束后结算奖励
  ├── 门派切磋 → 两名弟子互相对练，均获经验
  └── 设施升级 → 花费金币解锁/升级宿舍/练功房/兵器架/银庄/药堂

次日后进入下一天循环
```
